折线图控件使用说明

			            	
一、使用前准备

			            	
1.折线图控件需要依赖angular-flot模块，且需要加入以下js文件：jquery.flot.js,jquery.flot.time.js,jquery.flot.tooltip.min.js,jquery.flot.spline.js,jquery.flot.resize.js,flot/jquery.flot.pie.js,flot/curvedLines.js,angular-flot.js

			            	
html中使用标签引入：<div flot class="flot-chart-content" dataset="flotChartData" options="flotLineOptions"></div>
			            	

			            	
二、使用

							
1.设置dataset属性，值为折线图的数据源。例：
	$scope.flotChartData= [
        {
            label: "bar",
            data: [
                [1, 34],
                [2, 25],
                [3, 19],
                [4, 34],
                [5, 32],
                [6, 44]
            ]
        }
    ];

							
2.设置options属性，定义了折线图的配置项， 例：
	$scope.flotLineOptions={

        series: {
            lines: {
                show: true,
                lineWidth: 2,
                fill: true,
                fillColor: {
                    colors: [
                        {
                            opacity: 0.0
                        },
                        {
                            opacity: 0.0
                        }
                    ]
                }
            }
        },
        xaxis: {
            tickDecimals: 0
        },
        colors: ["#1ab394"],
        grid: {
            color: "#999999",
            hoverable: true,
            clickable: true,
            tickColor: "#D4D4D4",
            borderWidth: 0
        },
        legend: {
            show: false
        },
        tooltip: true,
        tooltipOpts: {
            content: "x: %x, y: %y"
        }
    };

饼图控件使用说明

			            	
一、使用前准备

			            	
1.折线图控件需要依赖chart.js模块，且需要加入以下js文件：Chart.js,angular-chart.js

			            	
html中使用标签引入：
	<canvas id="doughnu1" class="chart chart-doughnut" height="55" width="55"
              	chart-data="data" chart-labels="labels" chart-options="option" chart-click="chartClick">
        </canvas>
			            	

			            	
二、使用

							
1.设置chart-labels、chart-data属性，值为饼图的数据源。例：
	$scope.labels = ["PC", "微信1", "APP"]
    $scope.data = [300, 500, 100]
	

			
2.设置options属性，定义了饼图的配置项， 例：
	$scope.option={
        animation:{
            responsive: false,
            animateScale:true
        }
    }	
							

							
3.设置chart-click属性，值为一个方法，返回值为当前点击饼图的对象
